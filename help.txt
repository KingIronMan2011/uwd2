UWD2 v{}

USAGE:
    {} [SUBCOMMAND]

SUBCOMMANDS:
    <no subcommand> On first run: prompts for admin, installs to %LOCALAPPDATA%\Programs\UWD2,
                    adds to user PATH, and registers a high-priority logon task. On subsequent
                    runs (or via the scheduled task): just removes the watermark.
    inject          Alias for the default behaviour above.
    remove          Elevates, removes the scheduled task, PATH entry, and install directory.
    help            Shows this menu
    about           Shows some information about UWD2
